# Local TUI Dev Tool (Hybrid Edition)

Công cụ quản lý task, automation chạy local với giao diện Jules Google + TUI UX.

## 1. Yêu cầu

- PowerShell 5.0+
- Font: **VT323** và **JetBrains Mono** (Cần cài đặt thủ công)

## 2. Cách chạy

1. Mở PowerShell tại thư mục `server`.
2. Chạy lệnh:
   ```powershell
   .\server.ps1
   ```
3. Mở trình duyệt truy cập: `http://localhost:2501`

## 3. Cấu trúc

- **Frontend**: Vanilla JS + CSS Grid + Variables (Jules Theme)
- **Backend**: Single-file PowerShell Server
- **Data**: JSON formatted
